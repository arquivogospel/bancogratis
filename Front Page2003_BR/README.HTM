Como usar este documento

Para exibir o arquivo Leiame na tela, maximize a janela do navegador. Para imprimi-lo, clique no botão Imprimir na barra de ferramentas.

Locais dos arquivos Leiame

Antes da instalação, os arquivos Leiame encontram-se no CD1, em FILES\PFILES\MSOFFICE\OFFICE11\LCID, onde LCID é o número referente ao idioma do produto. Depois da instalação, eles são encontrados no disco rígido, em C:\Arquivos de programa\Microsoft Office\Office11\LCID, onde LCID é o número referente ao idioma do produto.

Requisitos do sistema

Todas as edições do Microsoft Office na versão 2003 têm aproximadamente os mesmos requisitos mínimos de sistema.

ProcessadorPentium 233 Mhz ou superior; Pentium III recomendado
Sistema operacionalMicrosoft Windows 2000 Service Pack 3 ou posterior, ou Windows XP ou posterior (recomendado)
Memória64 MB de RAM (mínimo); 128 MB de RAM (recomendado)
Espaço em disco245 MB, incluindo 115 MB de espaço disponível no disco rígido que contém o sistema operacional. O uso desse espaço em disco varia de acordo com a configuração. Uma fonte de instalação local exige cerca de 2 GB de espaço no disco rígido durante a instalação. Além do espaço necessário para o Office, a fonte de instalação local que permanece nos computadores dos usuários requer até 240 MB de espaço no disco rígido.
MonitorResolução Super VGA (800 x 600) ou superior, com 256 cores
Unidade de discoUnidade de CD-ROM
Dispositivo apontadorMicrosoft Mouse, Microsoft IntelliMouse® ou outro dispositivo apontador compatível

Observação    Nenhuma das edições do Microsoft Office na versão 2003 pode ser executada nos sistemas operacionais Microsoft Windows Me, Windows 98 ou Windows NT. Se algum computador cliente estiver executando um desses sistemas operacionais, atualize-o antes de instalar o Microsoft Office.


Alguns programas do Microsoft Office System requerem serviços ou itens adicionais:

  • O reconhecimento de fala requer um processador Pentium II 400 Mhz ou superior.
  • O Microsoft Office InfoPath™ 2003 requer o Microsoft Internet Explorer 6.0 ou posterior.
  • As anotações manuscritas no Microsoft Office OneNote 2003 requerem o recurso para entrada à caneta do Tablet PC para capturar tinta digital.
  • Os recursos de colaboração avançados do Microsoft Office Outlook 2003 podem requerer o Microsoft Exchange Server 2003.
  • A funcionalidade de Internet requer acesso dial-up ou banda larga em separado.
