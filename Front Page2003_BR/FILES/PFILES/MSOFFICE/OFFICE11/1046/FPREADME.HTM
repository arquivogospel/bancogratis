Problemas com a Configuração de Segurança Avançada do Internet Explorer

Por padrão, o Microsoft Windows Server 2003 é instalado com a Configuração de Segurança Avançada do Microsoft Internet Explorer habilitada. Essas configurações podem evitar que alguns sites— incluindo sites criados com o Microsoft Office FrontPage 2003 — sejam exibidos de maneira adequada ou funcionem como esperado.

Para obter mais informações, visite o site do Atendimento Microsoft em http://support.microsoft.com.
